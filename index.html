React Native 与 Expo


React Native由Facebook开源， 是目前流行的跨平台移动应用开发框架之一，与React一样为组件式，熟悉前端的开发人员可以快速上手。


Expo包含一组工具、库和服务， 在React Native的基础上做了很多的设置与简化，可以快速上手项目同时为开发提供了多种便利的服务，也可以让我们使用自己喜欢的编辑器来写代码。
