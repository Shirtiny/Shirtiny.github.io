tips：

在xml的sql语句中，不能直接用大于号、小于号要用转义字符

解决方式：

转义字符：

  • &lt;是<，小于号

  • &gt;是>，大于号

  • &amp;是&，和

  • &apos;是’，单引号

  • &quot;是”，双引号

或

使用<![CDATA[]]>标记：

<![CDATA[ state <= 3 ]]>

不过要注意，<![CDATA[ ]]>标记的sql语句中的<where> <if>等标签不会被解析
